Student Performance Dashboard (Power BI)

Overview

This project is an interactive **Student Performance Dashboard** built using **Power BI**.
It analyzes student data to provide insights into academic performance, attendance patterns, and overall trends.

Objectives

* Analyze student performance across different subjects
* Identify top-performing students
* Understand the impact of attendance on marks
* Compare performance across teachers and classes
* Visualize gender distribution and marks trends

Dashboard Features

KPI Cards

* Average Marks
* Average Attendance
* Total Students
* Pass Percentage

Visualizations

* **Bar Chart** → Average Marks by Subject
* **Bar Chart** → Average Marks by Teacher
* **Table** → Top 10 Students by Average Marks
* **Scatter Plot** → Attendance vs Marks
* **Column Chart** → Marks Distribution
* **Pie Chart** → Gender Distribution

Interactive Filters (Slicers)

* Class
* Subject
* Teacher
* Gender

Key Insights

* Subjects with highest and lowest performance
* Relationship between attendance and marks
* Top-performing students
* Gender-wise student distribution
* Teacher-wise performance comparison

Tools Used

* Power BI Desktop
* DAX (Data Analysis Expressions)
* Excel (for dataset)

How to Use

1. Download the `.pbix` file from this repository
2. Open it using **Power BI Desktop**
3. Use slicers to filter data interactively
4. Explore different visuals for insights

This dashboard provides a comprehensive overview of student performance and helps in making data-driven academic decisions
